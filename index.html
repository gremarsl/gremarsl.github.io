Featured Project

Development of a Runtime Measurement System
A custom FPGA-based system for non-intrusive, high-resolution runtime measurement on embedded targets. Designed to replace software-based timing instrumentation with an external hardware observer — enabling continuous performance monitoring without altering the system under test.

"A complex system that works is invariably found to have evolved from a simple system that worked. A complex system designed from scratch never works and cannot be patched up to make it work. You have to start over with a working simple system."

— John Gall, The Systems Bible

This principle is at the core of how I approach embedded systems development. Start with something that works on real hardware, measure it, then iterate. The FPGA project above is a direct application of this idea — a simple, working measurement system that grew from a clear, minimal design.
